
# Ecommerce
> 電子商務網站

## Installation
install dependencies.
``` javascript
npm install
```

## Usage

### backend
You need to go to `Server` folder to start, 
The port will listen `7000`
``` javascript
npm start
```

### frontEnd
You need to go to `UI` folder to start,
The port is `8080`, so input `localhost:8080` in your browser.
``` javascript
npm start
```
